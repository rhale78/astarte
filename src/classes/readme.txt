Directory classes
------------------

This directory contains files that manage TYPE structures.
They include the following.

copytype.c   Functions for copying type structures.

printty.c    Functions for printing types.

typeutil.c   Utilities for types that are used by both the
             compiler and the interpreter.

t_tyutil.c   Utilities for types that are used only by the
             compiler.

type.c       Constructors for types, and definition of type structures
             for use by the compiler and interpreter.

type.doc     Discussion of the fields used by various kinds of
             TYPE node, and what those fields are for.
