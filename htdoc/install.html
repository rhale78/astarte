M0. Installing Astarte

Using the Austin 320 lab at ECU

Astarte is installed in the Austin 320 lab at ECU in directory /export/other/software/astarte. The executables are in /export/other/sparc/bin (for the Sparc architecture) and /export/other/x86/bin (for the Pentium architecture). Please do not install it yourself on these machines.

Installing it yourself

If you have a computer of your own on which you want to install Astarte, you can do so. Astarte currently runs only under flavors of Unix. It will run under Linux. You will need a C compiler, plus the gunzip, tar and make utilities. I recommend the GNU C compiler.

You can get the implementation by ftp from the computers in the Austin 320 lab. It is /export/other/software/astarte0-8-1.tar.gz.

Get that file. Be sure to get it in binary mode. You can also get the README file separately from the same directory.

Alternatively, using anonymous ftp, go to www.cs.ecu.edu/pub/astarte. In that directory you will find file astarte0-8-1.tar.gz and README.

Put the file in a directory where you would like to install Astarte. A suggestion is /usr/local/lib/astarte, but you can select any directory that you like. Unzip the file and untar it. For example, type 

    gunzip astarte0-8-1.tar
    tar xvf astarte0-8-1.tar
Then follow the instructions in the README file.
