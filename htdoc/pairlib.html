LC1. Operations on pairs, triples, etc.

  • Conversion to and from strings
  • Miscellaneous functions on pairs

    • Conversion to and from strings

    $
    $ converts a pair to a string by showing the pair in standard notation. For example, $(1,2) = "(1,2)".

    pull
    pull(s) extracts an ordered pair from the front of string s and returns an ordered pair (p,r) where p is the ordered pair that was extracted and r is the suffix of s that follows p in the string. For example, pull("(2,\"abc\") xyz") = ((2,"abc"), " xyz"). Any initial white space is skipped over.

    Extract
    A convenient way to get an ordered pair from a string is to put the string in a box and used Extract. For example, if box b contains "('a',99) abc" then 
      Extract [$(?p)] from b.
    binds x to ('a', 99) and leaves box b holding " abc".

    Miscellaneous functions on pairs

    To use the functions marked with (collect/pair.ast), you will need to import package collect/pair.ast.

    compare, <, >, <=, >=
    Comparisons between ordered pairs are lexicographic, with the left-hand member of the pair having priority. For example, (2,5) < (3,4).

    _pmap_
    (f _pmap_ g)(x,y) = (f x, g y).

    Example: (left _pmap_ right)((2,3), (4,5)) = (2,5). (collect/pair.ast)

    _tee_
    (f _tee_ g)(x) = (f x, g x).

    Example: If double(x) = 2*x and triple(x) = 3*x then (double _tee_ triple)(8) = (16,24). (collect/pair.ast)

    xpose
    xpose(x,y) = (y,x). (collect/pair.ast)

    xposef
    (xposef f)(a,b) = f(b,a).

    Example: (xposef -)(a,b) = b - a. (collect/pair.ast)
