LC7. Matrices

  • Building matrices
  • Selecting members

  • Modifying a matrix
  • Getting the size
  • Mapping
    • A matrix is a two-dimensional list. It is represented by a list of lists, where all of the member lists have the same length. For example, matrix 
        E=   a b c d
         e f g h
         i j k l
    is represented by the list of lists 
            [[a,b,c,d],
         [e,f,g,h],
         [i,j,k,l]]
    A matrix is allowed to have no members. An empty matrix always has no rows and no columns

    The following operations are available by importing package collect/matrix.ast.

    Building matrices

    _above_
    : (Matrix(`a), Matrix(`a)) -> Matrix(`a)

    x _above_ y is the matrix obtained by writing matrix x above matrix y. For example, using the matrix E shown above, E _above_ E is 

             a b c d
         e f g h
         i j k l
         a b c d
         e f g h
         i j k l
    It is required that x and y have the same number of columns. Otherwise, x _above_ y fails with exception sizeX.

    _aside_
    : ([`a], Matrix(`a)) -> Matrix(`a)

    c _aside_ x is the matrix obtained by adding column c to the left side of matrix x. For example, if c is list [u,v,w] and x is the matrix E above, then c _aside_ x is 

           u a b c d
       v e f g h
       w i j k l
    It is required that c have the same number of members as x has rows. Otherwise, c _aside_ x fails with exception sizeX.

    You can use _aside_ in patterns. For example, Match ?c _aside_ ?m = E. binds c to [a,e,i] and m to matrix 

           b c d
       f g h
       j k l

    _atop_
    : ([`a], Matrix(`a)) -> Matrix(`a)

    r _atop_ x is the matrix obtained by adding row to the top of matrix x. For example, if r is list [t,u,v,w] and x is the matrix E above, then r _atop_ x is 

             t u v w
         a b c d
         e f g h
         i j k l
    It is required that r have the same number of members as x has columns. Otherwise, r _atop_ x fails with exception sizeX.

    You can use _atop_ in patterns. For example, Match ?r _atop_ ?m = E. binds r to [a,b,c,d] and m to matrix 

          e f g h
      i j k l

    _beside_
    : (Matrix(`a), Matrix(`a)) -> Matrix(`a)

    x _beside_ y is the matrix obtained by writing matrix x to the left of matrix y. For example, using the matrix E shown above, E _beside_ E is 

             a b c d a b c d
         e f g h e f g h
         i j k l i j k l
    It is required that x and y have the same number of rows. Otherwise, x _beside_ y fails with exception sizeX.

    matrix
    [[`a]] -> Matrix(`a)

    matrix L is a matrix whose list of rows is given by list-of-lists L. All member lists of list L must have the same length, or matrix fails with exception sizeX.

    Selecting members

    #
    : (Matrix(`a), (Natural,Natural)) -> `a

    x # (i,j) is the member of matrix x at row i and column j, where indexing is from 1. If there is no such entry, it fails with exception subscriptX.

    #_
    : (Matrix(`a), (Natural,Natural)) -> `a

    x #_ (i,j) is the member of matrix x at row i and column j, where indexing is from 0. If there is no such entry, it fails with exception subscriptX.

    Selecting slices and submatrices

    ##
    : (Matrix(`a), ((Natural,Natural),(Natural,Natural)) -> Matrix(`a)

    x ## ((i,n),(j,m)) is the matrix obtained from x by deleting the first i rows and j columns, and then taking the first n rows and m columns from what is left. For example, using the matrix E from above, E##((1,2),(1,3)) is matrix 

        f g h
    j k l
    If i is larger than the number of rows or j is larger than the number of columns, then the result of x##((i,n),(j,m)) is an empty matrix. If n is larger than the number or remaining rows or m is larger than the number of remaining columns, then all of the rows or columns are taken.

    column
    : Natural -> Matrix(`a) -> [`a]

    column n x is the n-th column of matrix x, numbering from 1. For example, if x is the matrix E above, then column 2 x is list [b,f,j]. This fails with exception subscriptX if there is no such column.

    column_
    : Natural -> Matrix(`a) -> [`a]

    column_ n x is the n-th column of matrix x, numbering from 0. For example, if x is the array E above, then column_ 2 x is list [c,g,k]. This fails with exception subscriptX if there is no such column.

    columns
    : Matrix(`a) -> [[`a]]

    columns x is a list of the columns of matrix x. For example, if x is the matrix E above, then colunms x is list 

      [[a,e,i],
   [b,f,j],
   [c,g,k],
   [d,h,i]]

    diagLR
    : (Natural,Natural) -> Matrix(`a) -> [`a]

    diagLR (i,j) x is the left-to-right diagonal of two-dimensional list x that contains the entry at row i, column j (numbering from 1). For example, if x is the matrix E above, then diagLR(1,2) x is list [b,g,l], since the b is in row 1, column 2. If there is no such entry, then diagLR fails with exception subscriptX.

    diagLR_
    : (Natural,Natural) -> Matrix(`a) -> [`a]

    diagLR_ (i,j) x is the left-to-right diagonal of matrix x that contains the entry at row i, column j (numbering from 0). For example, if x is the matrix E above, then diagLR_(0,1) x is list [b,g,l], since the b is in row 0, column 1. If there is no such entry, then diagLR_ fails with exception subscriptX.

    diagRL
    : (Natural,Natural) -> Matrix(`a) -> [`a]

    diagRL (i,j) x is the right-to-left diagonal of matrix list x that contains the entry at row i, column j (numbering from 1). For example, if x is the matrix E above, then diagRL(1,4) x is list [d,g,j]. If there is no such entry, then diagRL fails with exception subscriptX.

    diagRL_
    : (Natural,Natural) -> Matrix(`a) -> [`a]

    diagRL_ (i,j) x is the right-to-left diagonal of matrix x that contains the entry at row i, column j (numbering from 0). For example, if x is the matrix E above, then diagRL_(1,2) x is list [d,g,j], since the g is in row 1, column 2. If there is no such entry, then diagRL_ fails with exception subscriptX.

    row
    : Natural -> Matrix(`a) -> [`a]

    row n x is the n-th row of matrix x, numbering from 1. For example, if x is the matrix E above, then row 2 x is list [e,f,g,h]. This fails with exception subscriptX if there is no such row.

    row_
    : Natural -> Matrix(`a) -> [`a]

    row_ n x is the n-th row of matrix x, numbering from 0. For example, if x is the matrix E above, then row_ 2 x is list [i,j,k,l]. This fails with exception subscriptX if there is no such row.

    rows
    : Matrix(`a) -> [[`a]]

    rows x is a list of the rows of matrix x. For example, if x is the matrix E above, then rows x is list 

            [[a,b,c,d],
         [e,f,g,h],
         [i,j,k,l]]

    splitAtLeft
    : Matrix(`a) -> ([`a], Matrix(`a))

    splitAtLeft x returns the pair consisting of the leftmost column of matrix x and the matrix obtained by removing the leftmost column of x. For example, if x is the matrix E above, then splitAtLeft x is pair ([a,e,i],A) where A is matrix 

        b c d
    f g h
    j k l

    splitAtTop
    : Matrix(`a) -> ([`a], Matrix(`a))

    splitAtTop x returns the pair consisting of the first row of matrix x and the matrix obtained by removing the first row of x. For example, if x is the matrix E above, then splitAtTop x is pair ([a,b,c,d],A) where A is matrix 

        e f g h
    i j k l

    Modifying a matrix

    #=
    : (Matrix(`a), ((Natural,Natural),`a)) -> Matrix(`a)

    x #= ((m,n),v) is matrix obtained from x by replacing x#(m,n) by v.

    #_=
    : (Matrix(`a), ((Natural,Natural),`a)) -> Matrix(`a)

    x #_= ((m,n),v) is matrix obtained from x by replacing x#_(m,n) by v.

    transpose
    : Matrix(`a) -> Matrix(`a)

    transpose(m) is the transpose of matrix m. (If t is the transpose of m, then rows of t are columns of m, and columns of t are rows of m.)

    Getting the size

    ncols
    : Matrix(`a) -> Natural

    ncols x is the number of columns in matrix x.

    nrows
    : Matrix(`a) -> Natural

    nrows x is the number of rows in matrix x.

    Mapping

    map
    : (`a -> `b) -> (Matrix(`a) -> Matrix(`b))

    map f m is the result of replacing each member x of matrix m by f(x).
