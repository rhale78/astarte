T6. Standard Data Abstractions

  • Abstractions that do not require imports
  • Abstractions that require imports
    • You can think of a data abstraction, or genus, as a collection of related species, but it is often more convenient to think of it as representing the shared characteristics and capabilities of the species that it contains. For example, genus RRING contains species that share the operations of addition, subtraction and multiplication.

    Abstractions that do not require imports

    The following abstractions are available to all packages. See the standard hierarchy for the relationships among these.
    ANY
    ANY contains all species.

    EQ
    EQ contains species on which an equality test can be done. (This excludes functions, for which equality is undecidable.)

    ORDER
    ORDER contains species on which order tests (such as <) can be done.

    RANKED
    RANKED contains species Natural and all enumerated species. Species in RANKED have an operation rank that converts them to a natural number.

    ENUMERATED
    ENUMERATED contains all enumerated species. These share some operations with natural numbers.

    INTEGER
    INTEGER contains species Natural and Integer.

    RATIONAL
    RATIONAL contains species Natural, Integer and Rational.

    REAL
    REAL contains species Natural, Integer, Rational and Real. These species share the operations of addition and multiplication, among some others.

    RATRING
    RATRING contains species Integer and Rational.

    RFIELD
    RFIELD contains species Rational and Real. These are the standard species that support addition, subtraction, multiplication and division.

    RRING
    RRING contains species Integer, Rational and Real. These species support addition, subtraction and multiplication.

    Abstractions that require imports

    The following are available by importing packages.

    NUMBER
    NUMBER includes all of the numeric species Natural, Integer, Rational, Real and Complex. Import math/math.ast.

    APPROX
    APPROX holds species Real and Complex. Here, operations such as exp, sin and sqrt are provided. Import math/math.ast.

    SEMIGROUP
    SEMIGROUP contains species that support addition. This includes Natural, Integer, Rational, Real, Complex and Modular. Import math/math.ast.

    GROUP
    GROUP contains species that support both addition and subtraction. This includes Integer, Rational, Real, Complex and Modular. Import math/math.ast.

    SEMIRING
    SEMIRING contains species that support both addition and multiplication. This includes Natural, Integer, Rational, Real, Complex and Modular. Import math/math.ast.

    RING
    RING contains species that support addition subtraction and multiplication This includes Integer, Rational, Real, Complex and Modular. Import math/math.ast.

    NRING
    NRING is the intersection of RING and NUMBER. Import math/math.ast.

    FIELD
    FIELD holds species Rational, Real, Complex and Modular. These are species where all four basic arithmetic operations (+, -, * and /) are available, with the usual rules of arithmetic applying. Import math/math.ast.

    NFIELD
    NFIELD is the intersection of FIELD and NUMBER. Import math/math.ast.

    ARRAY
    ARRAY contains boxes, lists of boxes, lists of lists of boxes, etc. Import collect/boxes.ast.

    NEW
    NEW contains boxes, pairs of boxes, pairs of pairs of boxes, etc. These are things that can be created with operation new. Import collect/boxes.ast.

    BOXES
    BOXES contains boxes, lists of boxes, pairs of boxes, and arbitrary mixtures of pairs and lists of boxes. These can be copied. Import collect/boxes.ast.
