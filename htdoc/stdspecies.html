T2. Standard Species

  • Species that do not require imports
  • Species that require imports

    • Species that do not require imports

    The following species are available to all packages.
    ()
    Species () contains just one member, which is also called (). This species is used in a variety of ways, such as the return type of a function that returns no useful information. Value () is also used as a "do nothing" operation.

    ASpecies
    Species can be made into values and treated like data. A species as a value has species ASpecies.

    Boolean
    Species Boolean contains two values: false and true.

    BoxFlavor
    Species BoxFlavor contains two values: shared and nonshared. It is used to indicate the flavor of a box. Testers shared? and nonshared? are also available.

    Char
    Species Char contains characters such as 'a' and '-'.

    Comparison
    Species Comparison contains three values: equal, less and greater. They are used for the result of function compare.

    CopyFlavor
    Species CopyFlavor contains three values: shared, nonshared and same. It is used to indicate the flavor of a box when copying. Testers shared?, nonshared? and same? are also available.

    ExceptionSpecies
    Species ExceptionSpecies holds exceptions. These are items that are associated with failures, and indicate the reason for failure.

    FileMode
    Species FileMode contains values used in selecting the mode with which a system file is opened for reading or writing.

    Infinity
    Species Infinity contains two values: infinity and minusInfinity. It is used for forming infinite lists. You can use predicates infinity? and minusInfinity? to test for values. For example, infinity?(infinity) = true, but infinity?(minusInfinity) = false.

    Integer
    Species Integer contains integers, including negative integers.

    Natural
    Species Natural contains nonnegative integers.

    Rational
    Species Rational contains rational numbers, stored as ratios of integers.

    Real
    Species Real contains real numbers, stored approximately.

    UnknownKey
    Species UnknownKey is used for controlling access to unknowns.

    Species that require imports

    The following species are available by importing the indicated library packages.
    Bitvec
    Species Bitvec holds infinite bit vectors. Import collect/bitvec.ast.

    Complex
    Species Complex holds complex numbers, stored as pairs of real numbers. Import math/complex.ast.

    Modular
    Species Modular holds numbers on which modular arithmetic is to be performed. Import math/modular.ast.

    Symbol
    A symbol is similar to a string, but equality comparisons are faster for symbols than for strings. Import logic/symbol.ast.
