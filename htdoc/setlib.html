    • Sets

    A set is an unordered collection without repetition. The library supports sets using three different representations.

    Linear sets: (species LinSet(T)) These are sets stored as lists of their members. These can be used for any member species that supports equality tests.

    Sparse sets: (species SparseSet(T)) These are sets stored as binary search trees. These can be used for any member species that has an order relation (<).

    Dense sets: (species DenseSet(T)) These are sets stored as bit vectors. These can be used for any ranked species, such as Natural, Char, or enumerated species.

    All three families LinSet, SparseSet and DenseSet are members of community SET, which in turn belongs to the more general community COLLECTION. COLLECTION also contains family List.

    To use sets, import one or more of the set implementations, collect/linset.ast, collect/sparsset.ast or collect/denseset.ast. The shared operations are described in collect/set.ast and collect/collect.ast, but you do not need to import those packages to use sets. To create a set, use constructor set on a list. For example, 

       Let s = set[1,2,3].
    creates the set {1,2,3}. Operations on sets are as follows. Each shows the package that defines it.

    Empty set

    emptySet
    : SET`s(EQ`a)

    emptySet is the empty set. (collect/set.ast)

    emptySet?
    : SET`s(EQ`a) -> Boolean

    emptySet(s) is true if s is the empty set. (collect/set.ast)

    Conversion to and from lists

    set
    : [EQ`a] -> SET`s(EQ`a)

    set(x) is the set of all members of list x. (collect/set.ast)

    Example: set[2] is the singleton set 2.

    linSet
    : [EQ`a] -> LinSet(EQ`a)

    Same as set, but makes a linear set. (collect/linset.ast)

    sparseSet
    : [ORDER`a] -> SparseSet(ORDER`a)

    Same as set, but makes a sparse set. Sparse sets use order relations to keep track of the set. (collect/sparsset.ast)

    denseSet
    : [EQ`a] -> DenseSet(EQ`a)

    Same as set, but makes a dense set. (collect/denseset.ast)

    toList
    : SET`s(EQ`a) -> [EQ`a]

    toList(s) is a list holding the members of set s. (collect/collect.ast)

    Example: toList(set[3,2]) = [2,3]. toList can be lazy (depending on the kind of set).

    eagerToList
    : SET`s(EQ`a) -> [EQ`a]

    eagerToList is similar to toList, but is always eager. It computes the entire list before returning. (collect/collect.ast)

    Membership testing and getting members

    _in_
    : (EQ`a, SET`s(EQ`a)) -> Boolean

    x _in_ s is true if x is a member of set s. (collect/collect.ast)

    Example: 5 _in_ set[5,6,7]) = true.

    each
    : SET`s(EQ`a) -> EQ`a

    each(s) returns each member of set s, in a stream.

    eachMixed
    : SET`s(EQ`a) -> EQ`a

    eachMixed(s) returns each member of set s, in a mixed stream.

    Cardinality

    size
    : SET`s(EQ`a) -> Natural

    size(s) is the number of members that s has. (collect/collect.ast)

    Example: size(set[5,6,7]) = 3.

    sizeAtLeast
    : Natural -> SET`s(EQ`a) -> Boolean

    sizeAtLeast n s is true if the size of set s is at least n. (collect/collect.ast)

    Example: sizeAtLeast 5 (set[5,6,7]) = false.

    Inserting and deleting members

    +/
    : (SET`s(EQ`a), EQ`a) -> SET`s(EQ`a)

    s +/ x is the result of adding x to s. (collect/set.ast)

    Example: set[5,6,7] +/ 8 = set[5,6,7,8],
    set[5,6,7] +/ 5 = set[5,6,7].

    You can use +/ in patterns. Pattern ?x +/ v matches a set s that contains v. It binds x to x -/ v. Pattern ?x +/ ?v binds v to each member of the target set s, returning each in a stream, and binds x to s -/ v.

    -/
    : (SET`s(EQ`a), EQ`a) -> SET`s(EQ`a)

    s -/ x is the result of deleting x from s. (collect/set.ast)

    Example: set[5,6,7] -/ 8 = set[5,6,7],
    set[5,6,7] -/ 6 = set[5,7].

    Getting the smallest and largest

    min
    : SET`s(ORDER`a) -> ORDER`a

    min(s) is the smallest member of set s. (collect/collect.ast)

    Example: min(set[5,6,7]) = 5.

    max
    : SET`s(ORDER`a) -> ORDER`a

    max(s) is the largest member of set s. (collect/collect.ast)

    Example: max(set[5,6,7]) = 7.

    extractMax
    : SET`s(ORDER`a) -> (ORDER`a, SET`s(ORDER`a))

    extractMax(s) = (x,s') where x is the largest value in set s, and s' is s -/ x. (collect/set.ast)

    Example: extractMax(set[5,6,7]) = (7, set[5,6]).

    extractMin
    : SET`s(ORDER`a) -> (ORDER`a, SET`s(ORDER`a))

    extractMin(s) = (x,s') where x is the smallest value in set s, and s' is s -/ x. (collect/set.ast)

    Example: extractMin(set[5,6,7]) = (5, set[6,7]).

    Boolean operations

    \/
    : (SET`s(EQ`a), SET`s(EQ`a)) -> SET`s(EQ`a)

    s \/ t is the union of s and t. (Note: \/ is made by typing \ followed by /.) (collect/set.ast)

    Example: set[5,6,7] \/ set[5,8,9] = set[5,6,7,8,9].

    /\
    : (SET`s(EQ`a), SET`s(EQ`a)) -> SET`s(EQ`a)

    s /\ t is the intersection of s and t. (Note: /\ is made by typing / followed by \.) (collect/set.ast)

    Example: set[5,6,7] /\ set[5,8,9] = set[5].

    -
    : (SET`s(EQ`a), SET`s(EQ`a)) -> SET`s(EQ`a)

    s - t is the difference of s and t. That is, it is the set of all members of s that are not also members of t. (collect/set.ast)

    Example: set[5,6,7] - set[5,8,9] = set[6,7].

    Comparing sets

    ==
    : (SET`s(EQ`a), SET`s(EQ`a)) -> Boolean

    s == t is true if sets s and t have the same members. (collect/set.ast)

    Example: set[5,6,7,7] == set[5,6,7] is true.

    _subset?_
    : (SET`s(EQ`a), SET`s(EQ`a)) -> Boolean

    s _subset?_ t is true if s is a (not necessarily proper) subset of t. (collect/set.ast)

    Example: set[5,6,7] _subset?_ set[5,8,9] is false,
    set[5,6,7] _subset?_ set[5,6,7,8,9] is true.

    Checking and selecting members for properties

    someSatisfy
    : (EQ`a -> Boolean) -> SET`s(EQ`a) -> Boolean

    someSatisfy f s is true if some member x of set s makes f(x) = true. (collect/collect.ast)

    Example: someSatisfy (?x => x > 6) (set[5,6,7]) is true.

    allSatisfy
    : (EQ`a -> Boolean) -> SET`s(EQ`a) -> Boolean

    allSatisfy f s is true if every member x of set s makes f(x) = true. (collect/collect.ast)

    Example: allSatisfy (?x => x > 6) (set[5,6,7]) is false.

    select
    : (EQ`a -> Boolean) -> SET`s(EQ`a) -> EQ`a

    select f s is some member x of set s such that f(x) = true. (collect/collect.ast)

    Example: select (?x => x > 6) (set[5,6,7]) = 7

    Mapping, filtering and folding

    map
    : (EQ`a -> EQ`b) -> SET`s(EQ`a) -> SET`s(EQ`b)

    map f (set[a,b,c]) = set[f(a), f(b), f(c)]. (collect/collect.ast)

    Example: map (?x => x^2) (set[5,6,7]) = set[25,36,49].

    MapProc
    : (EQ`a -> ()) -> SET`s(EQ`a) -> ()

    MapProc (P) (s) %MapProc applies procedure P to each member of set s. (collect/collect.ast)

    mapSuccesses
    : (EQ`a -> EQ`b) -> SET`s(EQ`a) -> SET`s(EQ`b)

    mapSuccesses is similar to map, but it does not include values f(x) where f(x) fails. (collect/collect.ast)

    filter
    : (EQ`a -> Boolean) -> SET`s(EQ`a) -> SET`s(EQ`a)

    filter f s is the set of all member x of s such that f(x) is true. (collect/collect.ast)

    Example: filter (?x => x > 6) (set[5,6,7,8,4]) = set[7,8].

    bpartition
    : (EQ`a -> Boolean) -> SET`s(EQ`a) -> (SET`s(EQ`a), SET`s(EQ`a))

    bpartition f s = (s1,s2) where s1 is the set of all members of s that make f return true, and s2 is the list of all members of s that make f return false. (collect/collect.ast)

    Example: bpartition (?x => x > 5) (set [2,3,9,3,8,10,4]) = (set[2,3,3,4], set[9,8,10]).

    fold
    : (EQ`a,(EQ`a,EQ`a)->EQ`a) -> SET`s(EQ`a) -> EQ`a

    fold (i,+) (set[a,b,c]) = a + b + c. It is required that + be a commutative, associative binary operator, and that i be an identity for it. (collect/collect.ast)

    Example: fold (1,*) (set[5,6,7]) = 5*6*7 = 210.
