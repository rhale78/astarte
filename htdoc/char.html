    • Characters

    Astarte currently uses ascii characters. Characters have species Char. The following are abbreviated descriptions. See lang/char.ast for details.

    Most of what is described here is standard. For some of these operations, however, you must import lang/char.ast. They are marked as coming from lang/char.ast below.

    Character constants

    'a'
    The character a.

    '\n'
    The newline character.

    eol
    The newline character. This can only be used if you import lang/char.ast.

    '\t'
    The tab character.

    '\\'
    The backslash character.

    '\{23}'
    The character whose code is 23 (decimal).

    Classifying characters

    digit?
    digit?(c) is true if c is a decimal digit. lang/char.ast

    letter?
    letter?(c) is true if c is a letter. lang/char.ast

    lower?
    lower?(c) is true if c is a lower case letter. lang/char.ast

    printable?
    printable?(c) is true if c is a printable character. lang/char.ast

    upper?
    upper?(c) is true if c is an upper case letter. lang/char.ast

    white?
    white?(c) is true if c is a blank, tab or newline. lang/char.ast

    Case conversions

    toLower
    toLower(c) is the lower case equivalent of c (or is just c if c is not an upper case letter). lang/char.ast

    toUpper
    toUpper(c) is the upper case equivalent of c (or is just c if c is not a lower case letter). lang/char.ast

    Conversion to and from natural numbers

    char
    char(n) is the character with ascii code n.

    rank
    rank(c) is the ascii code of character c.

    Conversion to string

    If c is a character then $(c) is a string that shows c as a character constant. For example, $('a') = "'a'" and $('\n') = "'\\n'".

    If you want to convert a character c to a string that is one character long, build a list. For example, ['a'] is the string "a", and [c] is the string that only contains character c.

    Getting a character from a string

    Function pull: String -> (Char,String) removes a quoted character constant from the start of a string. For example, pull(" 'a' and more") returns the ordered pair ('a', " and more").

    You can get a character constant from a string using Extract and $ to extract a character from the front of a string in a box. For example, if box b holds string "'z' nnn", then performing 

       Extract [$(?c)] from b.
    binds c to character 'z' and leaves box b holding " nnn".

    If you want to read the first character from a string, without quotes, use 

       Extract [[?c]] from b.
    This will get the first character of the contents of box b, regardless of what it is. To get the first nonblank character, use 
       Extract [white ++ [?c]] from b.
