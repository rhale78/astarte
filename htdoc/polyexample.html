T8. More Examples of Polymorphism

Here are a few examples of functions with polymorphic types, and whose types use variables that do not range over ANY. Other examples can be found in under T5. Polmorphism.

/ : SFIELD`a -> SFIELD`a
/(x) is the reciprocal of x. It can take a rational number and produce a rational number or it can take a real number and produce a real number.

+ : (REAL`a, REAL`a) -> REAL`a
The addition function takes a pair of numbers and produces their sum.

< : (ORDER`a, ORDER`a) -> Boolean
The less-than function takes a pair of values and produces a boolean result (telling if the first is smaller than the second).

== : (EQ`a, EQ`a) -> Boolean
This function takes two values of the same species and tells whether they are equal.

==? : (EQ`a, EQ`b) -> Boolean
This unconstrained equality function takes two values (possibly of different species) and tells whether they are equal.

divide: (INTEGER`a, Natural) -> (INTEGER`a, Natural)
The divide function takes a pair (i,n) and returns the pair (q,r), where q is the quotient when you divide i by n and r is the remainder. Notice that n must be a natural number, but i can be either a natural number or an integer. Also, q must have the same species as i.
