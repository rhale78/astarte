L12. Pseudo-Random Numbers

Package misc/random.ast provides a pseudo-random number generator. The following functions are provided.

SetRandomSeed
: String -> ()

SetRandomSeed(s). should be used to initialize the pseudo-random number generator with a seed value of truly random digits. The seed s should be a string of at least 44 decimal digits. For each million bits generated, another 44 digits are consumed from this string.

Example:
SetRandomSeed "48399210909178352761873368291736286982647720".

Randomize
: () -> ()

Randomize. uses the system clock to set the seed of the pseudo-random number generator.

Example: Randomize.

coinToss
: () -> Natural

coinToss() returns either 0 or 1, drawn from the pseudo-random sequence. Each call to coinToss gets the next value in the sequence.

Example: Let c = coinToss().

random
: Natural -> Natural

random(n) is a pseudo-random integer uniformly distributed from 0 to n-1.

Example: Let r = random(30).

randomRange
: (Natural,Natural) -> Natural

randomRange(m,n) is a pseudo-random integer uniformly distributed from m to n.

Example: Let r = randomRange(1,20).
