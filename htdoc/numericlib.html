L4. Numeric Species and Operations

    • Numeric Species

    The Astarte library has four primary standard numeric species, and two primary nonstandard ones. The species are as follows.

    Natural
    Natural numbers 0, 1, 2, 3, ...

    Integer
    Integers ..., -2, -1, 0, 1, 2, ...

    Rational
    Rational numbers, stored exactly as a ratio of two integers.

    Real
    Real numbers, stored approximately.

    Complex
    Complex numbers, stored approximately.

    Modular
    Numbers on which modular arithmetic is done.
    Important note. There is a problem that occurs fairly frequently with natural numbers. When you use natural numbers, you cannot subtract, since the result might be negative. So if n has species Natural, then n-1 is not allowed. Write n--1 instead. Operator -- is absolute difference (x--y = |x-y|).

    Numeric data abstractions

    There is a hierarchy of genera for working with these numeric species. The standard hierarchy includes only the blue genera and species. If you import math/math.ast, then you get the full hierarchy, except species Modular and Complex. To get Modular, import math/modular.ast. To get Complex, import math/complex.ast. Smaller genera are to the right.

                               Modular         \_ Complex
    Many operations are shared by several of the species, and they are given polymorphic types. For example, the polymorphic type of the addition function, +, is (REAL`a,REAL`a) -> REAL`a if you do not import math/math.ast, and is extended to (SEMIGROUP`a,SEMIGROUP`a) -> SEMIGROUP`a if you do import math/math.ast. In the following, however, the operations that are available for each primary numeric species are listed without indicating their polymorphic types. See file math/stdnum.ast and math/math.ast for the polymorphic types.

    Typically, you cannot mix different primary species. For example, you can add two natural numbers, or add two integers, but you cannot add an integer and a natural number. See secondary numeric species for how to mix numeric species using secondary species.
