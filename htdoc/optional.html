L8. Optional Values

  • Basic functions
  • Other functions
    • You use optional values in situations where you might have a value and might not. An optional value that might be something of species T has species Optional(T).

    Basic functions

    noValue
    : Optional(`a)

    noValue is a non-value, as an optional value.

    noValue?
    : Optional(`a) -> Boolean

    noValue?(x) is true just when x is noValue.

    value
    : `a -> Optional(`a)

    value(x) is value x, as an optional value. You can use this in patterns.

    Example: value(20) is an optional number.

    value?
    : Optional(`a) -> Boolean

    value?(x) is true just when x is not noValue.

    unvalue
    : Optional(`a) -> `a

    unvalue(value(x)) = x. Expression unvalue(noValue) fails with exception testX.

    $
    : Optional(`a) -> String

    $(value(x)) = "value(S)" where S is $(x). $(noValue) = "noValue".

    Other functions

    optionalf
    : (`a -> `b) -> (`a -> Optional(`b))

    (optionalf f x) = value(f(x)) when f(x) succeeds, and is noValue when f(x) fails.
