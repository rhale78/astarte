T19. Abbrev Declarations

  • Creating abbreviation
  • Examples
  • Variables in abbreviations

    • Creating abbreviations

    Declaration 
           Abbrev N = T.
    indicates that identifier N is an abbreviation for type T. You can also use this to abbreviate a family, genus, community, or variable.

    Examples

           Abbrev Nat = Natural.
       Abbrev NUM = REAL.
       Abbrev A = `a.
       Abbrev B = (REAL`a, REAL`a).

    Variables in abbreviations

    The variables that you use in an abbreviation are not the same as any variables you create elsewhere, regardless of their names. For example, if you write 
           Abbrev A = `a.
       Abbrev B = `a.
    then A and B are different variables, in spite of the fact that both were given the name `a in thier declarations.

    All occurrences of an abbreviation in a given declaration contain the same variables, and abbreviations can refer to one another. For example, if you write 

           Abbrev A = `a.
       Abbrev B = (A,A).
       Abbrev F = B -> A.
       Abbrev F' = B -> `a.
       Abbrev F'' = A -> `a.
    then F is (`a,`a) -> `a, but F' is (`b,`b) -> `a and F'' is (`a -> `b).

    Abbreviations used in different declarations always have different variables in them. So using an abbreviation in one declaration cannot interfere with another declaration.
