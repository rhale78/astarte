LC4. Strings and Operations on Strings

A string is a list of characters (species [Char]). The list operations can therefore be used on strings. You can use String to abbreviate [Char].

You can create a string constant using double quotes. For example, "abc" is another way to write the list ['a', 'b', 'c'] (but in packed representation). To put a double quote inside such a constant, precede it by a backslash.

A normal string constant is not allowed to contain a line break. (It can contain the escape sequence \n, but not a true line break.) Multiline string constants are written as in the following example. All lines following the first %" whose first non-white characters are %" are concatenated together, each followed by a newline character. 

    %" This is a long sting constant.
    %" It can go on for several
    %" lines, and forms a single string.
If the last character of a line is a backslash, the newline at the end of that line is suppressed. For example, here is a single line string constant, with only a newline character at the end. 
    %" This is a long sting constant.\
    %" It can go on for several\
    %" lines, and forms a single string.
