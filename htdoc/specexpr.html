T4. Complex Species

  • Species Constructors
  • Examples

    • Species Constructors

    You can produce complex species from simple species using the following constructs.
    (...)
    You can use parentheses as much as you like in a complex species to group things.

    (A,B)
    Species (A,B) contains ordered pairs (x,y) where x has species A and y has species B. For example, species (String, Natural) contains pairs ("sheep", 2) and ("goat", 4). Species (A,B) is called a cartesian product species. The parentheses are required; you cannot write simply A,B.

    Notation (A,B,C) abbreviates (A,(B,C)), notation (A,B,C,D) abbreviates (A,(B,(C,D))), etc. So (true,2,"six") has species (Boolean, Natural, String).

    A -> B
    Species A -> B contains functions with domain A and codomain B. That is, they take a parameter of species A and produce a result of species B. Operator -> associates to the right, so A -> B -> C is understood to mean A -> (B -> C).

    F T
    If F is a parameterized species and T is a species, then F T is a species. For example, List(String) is a species. One of the members of that species is list ["one", "two", "three"]. This operation has higher precedence than ->. So F A -> B is understood to mean (F A) -> B.

    [T]
    [T] abbreviates List(T).

    <:T:>
    <:T:> abbreviates Box(T).

    String
    String abbreviates [Char].

    Examples

    The type building constructs can be combined in arbitrary ways. Here are a few examples.
    (Natural, Natural) -> Natural
    is the species of the addition function that takes a pair of natural numbers and produces a natural number as its result.

    ([String], Natural) -> String
    is the species of a function whose parameter must be an ordered pair consisting of a list of strings and a natural number, and that produces a string as its result. An example of such a function is one that takes pair (x,n) and produces the n-th member of list x.

    [String] -> (String, [String])
    is the species of a function that takes a list of strings as a paramter and produces an ordered pair consisting of a string and a list of strings.

    [Natural -> Natural]
    is the species of a list of functions, each of which takes a natural number and produces a natural number.

    Natural -> Natural -> Natural
    Keep in mind that -> associates to the right. So this species is read Natural -> (Natural -> Natural). A function of this species takes a natural number as its parameter and produces, as its result, a function of species Natural -> Natural.
