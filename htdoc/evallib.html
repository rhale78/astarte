L13. Controlling Evaluation

The following functions can be used to control how evaluation is done. See lazy evaluation for a discussion of lazy evaluation.

force
: `a -> `a

force(e) evaluates e up to the point where its top level structure is known, and returns e. For example, if e is a list, then force(e) will evaluate e until it is known either that e is the empty list or that e is a list that has a head and tail. The head and tail might not be evaluated.

forceAll
: `a -> `a

forceAll completely evaluatee a value, evaluating all lazy parts. For example, if the value is a list, then the entire list, including all of its members, will be evaluated. forceAll(e) returns the value of e.

forceList
: [`a] -> [`a]

forceList(e) evaluates the entire structure of list e, but not the members of the list. It returns e.
