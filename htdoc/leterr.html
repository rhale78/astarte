Let for Define

If you make a recursive definition using Let instead of Define, you will get an error. For example, definition 
Let sum by
  case sum []       = 0
  case sum (?h::?t) = h + sum(t)
%Let
causes an error message complaining that identifier sum is undefined. Fix this by using Define. The correct definition is 
Define sum by
  case sum []       = 0
  case sum (?h::?t) = h + sum(t)
%Define
