T14. Extension Declarations

You can create your own new species, families, genera or communities, but you must do so within an extension. The form is 
      Extension
          declaration 
         ...
          declaration 
      %Extension

The following declarations must occur within an extension. They cannot be written outside an extension.

  • Species declarations
  • Abstraction declarations
  • Relate declarations
  • Expect species declarations

The following kinds of declarations may be used inside an extension, but can also occur outside extensions.

  • Abbrev declarations
  • Advise declarations
  • Default declarations
  • Description declarations
  • Missing declarations
