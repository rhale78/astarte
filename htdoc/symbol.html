L7. Symbols

A symbol is a value that is completely characterized by its name. Symbols are common in languages that are designed for symbolic computing, such as Lisp and Prolog.

A symbol is represented by its name, but the name is interned. That is, the name is put into a hash table, so that every occurrence of the symbol refers to the same occurrence of the string in memory. You get two benefits from this. First, you don't store multiple copies of the symbol's name. Second, equality comparisons of symbols is done by comparing pointers, which is faster than comparing names character by character.

A symbol has species Symbol in Astarte. You need to import package logic/symbol.ast to use symbols. Operations are as follows.

#
: String -> Symbol

If s is a string, then #s is the symbol whose name is s. You can also apply # to a list of strings, with #[a,b,c] = [#a, #b, #c].

Examples
#"wheat"
#["the", "moon", "is", "yellow"]

# (pattern)
: String -> Symbol

You can use # in a pattern, either applied to a string or to a list of strings.

Example: 

  Match #(?str) = sym.
binds str to the name of symbol sym.

symbolName
: Symbol -> String

symbolName(x) is the name of symbol x (a string).

Example: symbolName(#"toast") = "toast".

$
: Symbol -> String

If x is a symbol, then $(x) is a string that consists of the name of x, in quotes, preceded by a #.

Example: $(#"toast") = "#\"toast\"".

pull
: String -> (Symbol, String)

When pull has species String -> (Symbol,String), it extracts the description of a symbol from the front of a string. The description should be in the form produced by $. For example, pull("#\"toast\" 334") = (#"toast", " 334").

compare
: (Symbol, Symbol) -> Comparison

Comparison of symbols compares their names. You can use ==, =/=, >, <, >= and <= to compare symbols.

symlist
: String -> [Symbol]

symlist "abc def ghi" = [#"abc", #"def", #"ghi"]. In general, symlist breaks a string at white space, and then converts each piece to a symbol. Parts of the string that are surrounded by quotes are kept together. For example, symlist "\"abc def\" ghi" = [#"abc def", #"ghi"].
