Setting up your account

To use Astarte in the Austin 320 lab at ECU, put one of the following directories in your path.

  • For use on Sparcstations: /export/other/sparc/bin

  • For use on Dells: /export/stu/x86/bin
You are best off editing your .cshrc file. Find a line that looks like this. 
   set path = (~/bin $OWpaths $lpath /usr/ucb /bin /usr/bin /etc .)
Add the following after that line. 
   /bin/sparc
   if ( $status ) then
     setenv PATH ${PATH}:/export/other/x86/bin
   else
     setenv PATH ${PATH}:/export/stu/sparc/bin
   endif
Then reread your .cshrc file, by typing 
   source .cshrc
You only need to do the source command after modifying .cshrc, not at subsequent logins.
