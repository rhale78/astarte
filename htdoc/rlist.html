    • Random access lists (functional arrays)

    You can use packed lists (stored as arrays, and created by function pack) to achieve random access in lists. Operations such as subscripting into packed lists are efficient. However, they do not allow you to make changes to lists. Operations, #= (which replaces a member of a list), although relatively fast on a packed list, disrupts the structure of the list, so that it is no longer packed. The more of these that you do, the farther the list structure will stray from a simple array form.

    Package collect/funarr.ast provides a family FunArr of functional arrays. FunArr(T) is similar to [T]; it provides an alternate representation of lists. These support efficient updates (including operation #=) without the structure being damaged.

    FunArr supports rapid subscripting, modification by subscript and extension (adding something to the end of the array). It is used for cases where you want to use lists, but need to work on them in a way similar to the way you work on an array, making modifications to particular values. Family FunArr belongs to COLLECTION.

    The operations work identically to the corresponding operations on lists.

    Conversions between functional arrays and lists

    toFunArr
    : [`a] -> FunArr(`a)

    toFunArr(x) is a functional array holding the same members as list x.

    toList
    : FunArr(`a) -> [`a]

    toList(x) is the list whose members are the same as the members of functional array x. Function toList is lazy.

    eagerToList
    : FunArr(`a) -> [`a]

    This is the same as toList, but it is eager.

    Basic operations

    emptyFunArr
    : FunArr(`a)

    emptyFunArr is an empty functional array.

    nil?
    : FunArr(`a) -> Boolean

    nil?(x) is true just when x is an empty functional array.

    +/
    : (FunArr(`a),`a) -> FunArr(`a)

    x +/ z is array x with z added to the end. It takes time logarithmic in the length of x.

    Example: toFunArr[2,3] +/ 4 = toFunArr[2,3,4].

    allButLast
    : FunArr(`a) -> FunArr(`a)

    allButLast(x) is the result of removing the last member of functional array x. It takes time logarithmic in the length of x.

    #
    : (FunArr(`a), Natural) -> `a

    x # k is the k-th member of array x, numbering from 1. If k = 0 or x has fewer than k members then this operation fails with exception subscriptX. This operation takes time logarithmic in the length of x.

    Example: toFunArr[20,30,40] # 2 = 30.

    #_
    : (FunArr(`a), Natural) -> `a

    x #_ k is the k-th member of array x, numbering from 0. If x has fewer than k+1 members then this operation fails with exception subscriptX. This operation takes time logarithmic in the length of x.

    Example: toFunArr[20,30,40] # 2 = 40.

    #=
    : (FunArr(`a), (Natural, `a)) -> FunArr(`a)

    x #= (k,v) is the result of replacing the k-th member of array x by v, where indexing is from 1. If k = 0 or x has fewer than k-1 members then x #= (k,v) fails with exception subscriptX. If x has exactly k-1 members then x #= (k,v) is equivalent to x +/ v. This operation takes time logarithmic in the length of x.

    Example: toFunArr[20,30,40] #= (2,50) = toFunArr[20,50,40].
    toFunArr[20,30,40] #= (4,50) = toFunArr[20,30,40,50].

    #_=
    (FunArr(`a), (Natural, `a)) -> FunArr(`a)

    x #_= (k,v) is the result of replacing the k-th member of array x by v, where indexing is from 0. If x has fewer than k members then x #_= (k,v) fails with exception subscriptX. If x has exactly k members then x #_= (k,v) is equivalent to x +/ v. This operation takes time logarithmic in the length of x.

    Example: toFunArr[20,30,40] #_= (2,50) = toFunArr[20,30,50].

    length
    : FunArr(`a) -> Natural

    length(x) is the number of members of functional array x.

    Other operations

    ==
    : (FunArr(EQ`a), FunArr(EQ`a)) -> Boolean

    Two functional array are equal just when they contain exactly the same members, in the same order.

    map
    : (`a -> `b) -> FunArr(`a) -> FunArr(`b)

    This is similar to map for lists.

    Example: map (?x => x + 1) (toFunArr[20,30]) = toFunArr[21,31].
