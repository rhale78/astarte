T13. Standard Communities

A community is a set of families.

The following are available by importing packages.

COLLECTION
COLLECTION includes List, all of the set families and FunArr. Import collect/collect.ast.

SET
SET includes all of the set families DenseSet, SparseSet and LinSet. Import collect/set.ast.

TABLE
TABLE includes families LinTable and SparseTable. Import collect/table.ast.
The following are available to all packages. See the standard hierarchy for the relationships among these.

OPAQUE
Community OPAQUE contains all opaque families.

OPAQUE_EQ
Community OPAQUE contains all opaque families that are in genus EQ.

OPAQUE_ORDER
Community OPAQUE contains all opaque families that are in genus ORDER.

TRANSPARENT
Community TRANSPARENT contains all transparent families.

TRANSPARENT_EQ
Community TRANSPARENT contains all transparent families that are in genus EQ.

TRANSPARENT_ORDER
Community TRANSPARENT contains all transparent families that are in genus ORDER.
